Location -- create journal entries as individual files in `journal/` folder, named `YYYY-MM-DD-short-slug.md` (e.g. `journal/2026-03-07-stroke-opacity.md`)
Header metadata -- date, files modified/created/deleted with line numbers, so readers can navigate straight to the actual changes
Problem statement -- what needed to be solved and why
Solution implementation -- technical approach, key design decisions, code examples of critical changes
Testing/validation -- how the solution was verified
Development learnings -- mistakes the AI made that the user corrected, with exact quotes, so the same mistake doesn't repeat
