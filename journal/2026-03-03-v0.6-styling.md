## Problem Statement

The app had a single `strokeColor` and hardcoded `strokeWidth: 2`. Shapes had no fill, there was no opacity control, and the color picker was a bare native `<input type="color">`. v0.6 adds fill color, stroke width, opacity, an enhanced color picker with palette + hex input, and style memory via localStorage.

## Solution Implementation

**Two-layer fill rendering.** The rough/sketchy paths generated by roughjs are open stroke outlines — setting SVG `fill` on them colors the stroke lines, not the shape interior. The solution renders filled shapes as a composite: a clean SVG primitive (`<rect>` or `<ellipse>`) underneath for the fill, with the rough `<path>` on top for the stroke.

```tsx
// SceneRenderer.tsx — fill rendered as clean shape behind rough path
if (hasFill) {
  return (
    <g opacity={opacity !== 1 ? opacity : undefined} transform={...}>
      <FillShape obj={obj} />           {/* clean rect/ellipse with fill color */}
      <path d={obj.pathData} fill="none" stroke={obj.color} strokeWidth={strokeWidth} />
    </g>
  )
}
```

**Style memory.** A `loadStyles`/`persistStyles` pair reads/writes `{ strokeColor, fillColor, strokeWidth, opacity, fontSize }` to `localStorage` under key `doodler-styles`. The store initializes from saved values and each style setter persists the full style object.

**ColorPicker component.** A reusable popover with 16-color 2x8 palette grid, hex text input with validation, native `<input type="color">` fallback, and optional "No fill" button. Closes on outside click or Escape via `useEffect` event listeners.

## Development Learnings

**Rough path fill bug.** The initial implementation set `fill={fillColor}` directly on the rough `<path>` element. The user reported: "for rectangles instead of filling the rectangle area, it seems to color the stroke." Roughjs paths are open stroke outlines, not closed fillable shapes. The fix was to render a separate clean SVG primitive for the fill behind the rough path. This same pattern applies to SVG export and the active shape preview.
